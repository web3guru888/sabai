# Sabai Framework

> The open-source LINE Mini App framework for Thai commerce.

## Packages

| Package | Description |
|---------|-------------|
| `@sabai/core` | LIFF initialization, auth, i18n, and messaging for LINE Mini Apps |
| `@sabai/ui` | React components for Thai regulatory compliance |
| `@sabai/payments` | Payment integrations for Thailand (LINE Pay, PromptPay, Omise) |
| `create-sabai-app` | CLI scaffolding tool for Sabai projects |

## Apps

| App | Description |
|-----|-------------|
| `@sabai/e-commerce` | Reference e-commerce LINE Mini App |

## Getting Started

```bash
pnpm install
pnpm build
pnpm dev
```
